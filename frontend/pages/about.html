About RubricAI
AI-powered evaluation platform for student interview assessment
Project Overview

RubricAI evaluates student interview transcripts against a 34-indicator behavioral rubric. Built for CPS LEARN Lab at Northeastern University, it provides consistent, evidence-based scoring — replacing manual, subjective review with scalable AI-assisted assessment.

The tool evaluates student communication, critical thinking, and professional agency skills as demonstrated in interview simulations. All evaluations are grounded in research-backed rubric criteria and validated by instructors.

How Levels Are Set
1. Claude reads the rubric thresholds
2. Finds behavioral evidence in transcript
3. Matches evidence to Level 1, 2, 3, or 4
4. Returns score + confidence + quotes
5. Low confidence triggers instructor review
Confidence Thresholds
80–100% High — clear evidence found
60–79% Medium — some ambiguity
Below 60% Low — flagged for instructor
