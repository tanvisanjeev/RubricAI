About RubricAI
AI-powered rubric evaluation system for educational research

Project Overview

RubricAI evaluates student interview performance against research-backed behavioral rubrics. Developed for CPS LEARN Lab at Northeastern University to assess 300 students across 4 interviews each, processing 1,200 total transcripts with evidence-based AI evaluation.

Current Capabilities

  • Evaluates "Asks clear, open-ended questions" indicator (Levels 1-4)
  • Automated question extraction and classification from transcripts
  • Evidence-based justifications with AI confidence scoring
  • Instructor validation workflow with override functionality
  • Real-time processing with progress tracking

Development Roadmap

  • Phase 1 (Complete): Indicator 1 operational with test data validation
  • Phase 2 (4 weeks): Communication domain (10 indicators)
  • Phase 3 (4 weeks): Critical Thinking domain (12 indicators)
  • Phase 4 (4 weeks): Professional Agency domain (12 indicators)

Technical Architecture

  • Backend: Python FastAPI with SQLite database
  • AI Engine: Groq API (Llama 3.3 70B Versatile)
  • Frontend: Responsive web application
  • Evaluation Method: Deductive rubric-based assessment

Research Foundation

All indicators grounded in communication theory (Kvale & Brinkmann, 2009), critical thinking frameworks (Facione, 1990; Paul & Elder, 2006), and professional development research (Bandura, 1997; Dweck, 2006). Rubric criteria operationalize observable behaviors with quantifiable thresholds.
