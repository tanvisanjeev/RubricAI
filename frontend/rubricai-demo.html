About RubricAI
AI-powered assessment system for educational research

Project Overview

RubricAI evaluates student interview performance against research-backed behavioral rubrics. Developed for CPS LEARN Lab at Northeastern University to process 1,200+ interview transcripts with evidence-based AI evaluation.

Current Capabilities

  • Evaluates "Asks clear, open-ended questions" indicator (Level 1-4)
  • Counts and classifies student questions from transcripts
  • Provides evidence-based justifications with confidence scores
  • Extracts specific examples for instructor validation

Development Roadmap

  • Phase 1 (Complete): Indicator 1 operational with test data validation
  • Phase 2 (In Planning): Complete Communication domain (10 indicators)
  • Phase 3 (Future): Critical Thinking domain (12 indicators)
  • Phase 4 (Future): Professional Agency domain (12 indicators)

Technology Stack

  • Backend: Python FastAPI + Groq AI (Llama 3.3 70B)
  • Frontend: HTML/CSS/JavaScript (deployable to Vercel)
  • Evaluation: Research-backed rubric criteria with behavioral thresholds
