# RubricAI

AI-Powered Interview Skills Assessment Platform

Evaluates student interview performance against research-backed rubrics using AI.

## Features
- Upload interview transcripts (CSV)
- AI evaluation against rubric criteria (Levels 1-4)
- Evidence extraction and justification
- Instructor review dashboard

## Tech Stack
- Backend: Python FastAPI + Groq API
- Frontend: HTML + JavaScript
- Deployment: Vercel

## For LEARN Lab
Designed to evaluate 300 students × 4 interviews = 1,200 transcripts against communication rubrics.
